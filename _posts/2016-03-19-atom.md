[Atom editor](http://atom.io/): A hackable text editor for the 21st Century.

## Install

```bash
brew cask install atom
```

## Plugins for c/c++

* clang-format
* file-icons
* autocomplete-clang
* linter
* linter-clang
* atom-beautify
* terminal-plus
* atom-ctags
